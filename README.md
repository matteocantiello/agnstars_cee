# `agn_cee` — Compact-object mergers inside stars in AGN disks

Reproducibility package for

> **Signatures of Compact Object Mergers Inside Stars in AGN Disks**
> Cantiello et al. (in prep.)

This repository contains everything needed to reproduce the figures and the quantitative results of
the paper: a small, self-contained Python package (`agn_cee`), the input data, a script to regenerate
the figures, a test suite, and an annotated Jupyter notebook that walks through every figure.

The paper studies what happens when compact objects and binary black holes (BBHs) are captured by — and
merge inside — very massive, long-lived (“immortal”) stars embedded in AGN disks. The treatment is
deliberately analytic / order-of-magnitude: only a 3D self-gravitating calculation can resolve the deep
core, so the package computes timescales, energy budgets, and the $\mathcal{O}(1)$ gravitational-wave
signature rather than full waveforms.

## Installation

Python ≥ 3.9 with NumPy, SciPy, and Matplotlib. From the repository root:

```bash
pip install -r requirements.txt
```

Tested with Python 3.10, numpy 2.2.5, scipy 1.13.1, matplotlib 3.9.0.

## Usage

Run everything **from the repository root** (the data path is resolved relative to the working
directory).

- **Reproduce the figures + numbers interactively:** open `paper_figures.ipynb`. Each figure has a short
  markdown explanation (physical input + references) followed by the cell that produces it; the final
  section recomputes the numbers quoted in the paper.

  ```bash
  jupyter lab paper_figures.ipynb     # or: jupyter notebook
  ```

- **Regenerate all figures non-interactively:**

  ```bash
  python make_figures.py              # writes figures/fig*.pdf
  ```

- **Run the tests:**

  ```bash
  pytest tests/                       # 24 tests (physics + SNR calibration)
  ```

## Data

- `data/14547_cs10_RHO_3d-17_profile20.data` — MESA profile of the fiducial immortal star
  ($M_\star\simeq773\,M_\odot$), from the AGN-star models of Cantiello, Jermyn & Lin (2021). Loaded by
  `agn_cee.structure` and stitched to an analytic accretion stream out to the Bondi radius.
- `data/O3-L1-C01_..._sensitivity_strain_asd.txt` — representative LVK O3 (LIGO Livingston) strain
  amplitude-spectral-density curve, used for the LVK noise curve and SNR ($S_n=\mathrm{ASD}^2$). The
  LISA (Robson, Cornish & Liu 2019) and DECIGO (Yagi & Seto 2011) curves are analytic and defined in
  `agn_cee.detectors`.

## The `agn_cee` package

| Module | Contents |
|--------|----------|
| `constants.py` | CGS physical constants |
| `physics.py` | drag forces, BHL accretion, GW luminosity/frequency/strain, Peters inspiral, Eddington limits |
| `structure.py` | build the immortal-star + accretion-stream model from a MESA profile |
| `inspiral.py` | Case A: single-CO spiral-in (angular-momentum-loss) and destruction energetics |
| `bbh.py` | Case B: internal BBH hardening rates, merger time, dephasing |
| `detectors.py` | LISA / DECIGO / LVK noise curves and sky-averaging factors |
| `observability.py` | matched-filter SNR, characteristic strain, horizon distance |
| `rates.py` | migration time, encounter rate, volumetric merger rate |
| `figures.py` | Fig. 1–3, 7 |
| `review.py` | Case B / GW figures (Figs. 4–6, 8, 9) and the robustness diagnostics (sound-speed prescriptions, hardening model ladder, energy deposition, measurable dephasing) |
| `plotstyle.py` | shared Matplotlib style and color palette |
